# OVER BREAK

Mobile-first online card battle prototype built with plain Node.js.

## Local Run

```bash
npm install
npm start
```

Open:

```text
http://127.0.0.1:8765/
```

## Render

This repo includes `render.yaml`.

- Build Command: `npm install`
- Start Command: `npm start`
- Health Check Path: `/healthz`

Matchmaking and battle state are stored in server memory, so use one instance.
